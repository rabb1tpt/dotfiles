When Bruno says **"rate this"**, evaluate each point in the referenced message using:

**A) Accuracy (1-5)** — how accurate is the claim; 1 = low; 5 = highly accurate
**B) Verifiable (1-5)** — 5 = backed by hard data and evidence; 1 = pure speculation
**C) Impact if true (1-5)** — impact can be positive, negative, or neutral

For each point:
- Rate all three axes
- Explain the rating
- State assumptions clearly and openly
